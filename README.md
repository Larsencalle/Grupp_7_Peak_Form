# Grupp_7_Peak_Form
Grupp 7 repository

Instruktioner för att granska koden:

1. Gå till vårt GitHub-repository: https://github.com/Larsencalle/Grupp_7_Peak_Form

2. Vår kod är i en tidig frontend-fas. Klicka på "Releases" / "Tags" till höger på sidan och välj taggen v1.0-KG1 för att se det som gäller för denna granskning.

3. Koden kräver ingen installation eller databas just nu. För att testa den, ladda ner filerna och öppna index.html direkt i en valfri webbläsare.
